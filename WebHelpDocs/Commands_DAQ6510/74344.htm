channel.setcommonside()

This function sets up the specified slot to have a common side path.


This command is only applicable for cards that support commonside measurements, such as the Keithley Instruments 7701 module.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Function

Yes

 

 

 

Usage

channel.setcommonside("slot", state)

slot

A string that lists the slot number; can be allslots, slot1, or slot2

state

channel.ON or channel.OFF

Details

Set commonside ohms before setting the function.

Example

channel.open("allslots")

channel.setcommonside("slot1", channel.ON)

channel.setdmm("slot1", dmm.ATTR_MEAS_FUNCTION, dmm.FUNC_4W_RESISTANCE)

channel.close("101")

print(dmm.measure.read())

channel.close("132")

print(dmm.measure.read())

printbuffer(1, defbuffer1.n, defbuffer1)

channel.open("allslots")

Set the channels on slot 1 to have a common side path.

Close channel 101.

Make a measurement.
