channel.getcommonside()

This function lists the commonside setting for the specified slot.


This command is only applicable for cards that support commonside measurements, such as the Keithley Instruments 7701 module.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Function

Yes

 

 

 

Usage

state = channel.getcommonside("slot")

state

The state of the slot:

  • Common side disabled: 0
  • Common side enabled: 1
  • No card installed or card does not support commonside measurements: nil

slot

A string that lists the slot number; can be allslots, slot1, or slot2

Example

print(channel.getcommonside("slot1"))

Check the settings of channels 1 to 10 on slot 1. If common side is enabled, return is similar to:

[1]=1, [2]=1, [3]=1, [4]=1, [5]=1, [6]=1, [7]=1, [8]=1, [9]=1, [10]=1
