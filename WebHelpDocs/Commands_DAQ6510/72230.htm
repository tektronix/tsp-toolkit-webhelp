channel.getcountinterval()

This function returns the interval that is presently set for storage of channel relay closure counts.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Function

Yes

 

 

 

Usage

update = channel.getcountinterval()

update

The count update interval:

  • 1 minute: channel.COUNT_INTERVAL_1M
  • 5 minutes: channel.COUNT_INTERVAL_5M
  • 10 minutes: channel.COUNT_INTERVAL_10M
  • 15 minutes: channel.COUNT_INTERVAL_15M
  • 30 minutes: channel.COUNT_INTERVAL_30M
  • 60 minutes: channel.COUNT_INTERVAL_60M
  • 24 hours: channel.COUNT_INTERVAL_24H

Example

print(channel.getcountinterval())

If the interval was set to 24 hours, the return is:

channel.COUNT_INTERVAL_24H

Also see

channel.setcountinterval
