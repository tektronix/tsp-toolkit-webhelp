Simulated reference junction

The simulated reference temperature for the DAQ6510 can be set from 0 °C to 65 °C. The DAQ6510 measures the input voltage and factors in the simulated reference temperature to calculate the temperature reading at the thermocouple.

An example of a simulated reference junction is an ice bath. The input wire to thermocouple wire connections are immersed (but electrically isolated) in the ice bath, and the user enters the 0 °C simulated reference temperature into the DAQ6510.


The most accurate temperature measurements are achieved by using a simulated reference junction using an ice-point reference.
