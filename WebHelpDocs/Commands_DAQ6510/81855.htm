Channel naming

In the DAQ6510, channels are named using the slot number and the channel number. The slot number is the number of the slot on the rear panel where the card is installed. The channel assignment is formatted as SCH, where:

  • S is the slot number
  • CH is a two-digit channel number

For example:

  • 101 = slot 1, channel 1
  • 210 = slot 2, channel 10

If you are sending commands from a remote interface, you include the three-digit channel name, slot1, slot2, or allslots in the channel list parameter for the commands.

In TSP commands, the channel numbers are in quotes. To designate multiple individual channels, separate the channels with commas. In the following example, the command opens channels 1 and 3 on the card in slot 1.

channel.open("101, 103")

To designate a range of channels, separate the first and last channel number with a colon. The range can be from lowest to highest numbered channel or highest to lowest. The following example sets channel 1 to 9 on slot 1 to the DC voltage measurement function and then creates a scan with those channels.

channel.setdmm("101:109", dmm.ATTR_MEAS_FUNCTION, dmm.FUNC_DC_VOLTAGE)

scan.create("101:109")

Some commands allow you to set all channels in a slot or all channels in the instrument. In this case, you can send slot1 to set all channels in slot 1, slot2 for all channels in slot 2, or allslots for all channels in all slots. For example, to set all channels on slot 1 to measure voltage, you can send:

channel.setdmm("allslots", dmm.ATTR_MEAS_FUNCTION, dmm.FUNC_DC_VOLTAGE)

When you send slot1, slot2, or allslots to a slot that contains channels that cannot accept that setting, the instrument generates an error, but the change is made to all channels for which the setting is allowed.
