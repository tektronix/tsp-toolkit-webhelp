channel.getlabel()

This function returns the label associated with the specified channel.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Function

Yes

 

 

 

Usage

label = channel.getlabel("channelNumber")

label

A string that lists the comma-delimited label for the channel specified in channelNumber

channelNumber

A string that lists the channel to query for the associated label

Details

Returns nil if no label is set.

Example

channel.setlabel('1', 'First')

channel.setlabel('2', 'Second')

channel.setlabel('3', 'Third')

print(channel.getlabel('1'))

print(channel.getlabel('2'))

print(channel.getlabel('3'))

Set the labels to First, Second, and Third.

Return the new labels.

Output:

First

Second

Third
