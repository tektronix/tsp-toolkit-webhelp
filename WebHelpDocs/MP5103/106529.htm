localnode.model

This attribute stores the model number of the instrument.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Attribute (R)

Yes

Not applicable

Nonvolatile memory

Not applicable

Usage

model = localnode.model

model

The model number of the mainframe

Details

This attribute stores the model number of the mainframe.

To retrieve the model number of a module, use slot[Z].model.

Example

print(localnode.model)

Outputs the model number of the local node.

Example output:

MP5103
