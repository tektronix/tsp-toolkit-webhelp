lan.restoredefaults()

This function resets the LAN commands to default values.

This command is run when lan.reset() is sent.

If you are using TSP Toolkit, set lan.rawsocket.access and lan.telnet.access to localnode.ENABLE.

To apply the default settings, run lan.applysettings() after lan.restoredefaults().

This function resets the access to require a password for communications over LAN for lan.hislip.access, lan.rawsocket.access, and lan.telnet.access. The lan.restoredefaults() function does not reset the LAN password. The user.password.change() attribute controls the LAN password, which can be reset with lan.reset().

Example

lan.restoredefaults()

lan.applysettings()

Restores the LAN defaults.
