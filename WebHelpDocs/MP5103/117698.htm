status.reset()

This function resets all bits in the mainframe status model.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Function

Yes

 

 

 

Usage

status.reset()

Details

This function clears all status data structure registers (Enable, Event, NTR, and PTR) to their default values. For information about these registers, refer to Status register set contents.

This command does not affect the status model of the installed modules.

Example

status.reset()

Resets the status model of the mainframe.
