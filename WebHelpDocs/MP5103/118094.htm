login()

This command logs a user in to the mainframe and allows remote access.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Function

Yes

 

 

 

Usage

login("password")

password

A string that represents the login password; the default value is no password

Details

This command logs a user into the mainframe and allows remote access. A user remains logged in until the instrument is reset or the logout command is sent.

The password can be added or changed with the user.password.change command.

Logins persist throughout mainframe and module firmware updates, or until the logout command is sent or the mainframe is rebooted.

Example

login

logout

Logs a user into and then out of the mainframe test system.
