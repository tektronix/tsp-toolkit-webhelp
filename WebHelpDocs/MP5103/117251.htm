trigger.timer[N].stimulus

This attribute specifies which event starts the timer.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Attribute (RW)

Yes

Instrument reset
Recall setup
trigger.timer[N].reset

Not saved

0

Usage

eventID = trigger.timer[N].stimulus

trigger.timer[N].stimulus = eventID

eventID

The event that triggers the timer delay

N

Trigger timer number: 1 to 8

Details

The event IDs that can be used for this command include mainframe-specific event IDs as well as event IDs for any installed modules capable of generating a trigger event. See ***Trigger event IDs*** to find the event IDs for your mainframe and specific modules.

Example

print(trigger.timer[1].stimulus)

Prints the event that starts a trigger 1 timer action.

Also see
