channel.getmode()

This function returns the present mode of digital and totalizer channels.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Function

Yes

 

 

 

Usage

mode = channel.getmode("channelList")

mode

Return string of a comma-delimited list of modes

channelList

String that specifies the channels to query, using normal channel list syntax

Details

For digital channels, the return is:

  • 8 if the channel is set to channel.MODE_INPUT
  • 9 if the channel is set to channel.MODE_OUTPUT

For totalizer channels, the return is:

  • 1 if the channel is set to channel.MODE_RISING_EDGE
  • 0 if the channel is set to channel.MODE_FALLING_EDGE
  • 3 if the channel is set to channel.MODE_RISING_EDGE_READ_RESET
  • 2 if the channel is set to channel.MODE_FALLING_EDGE_READ_RESET

For DAC channels, the return is 17.

This command is not available for other types of channels. If other channels are included in a range, they are ignored. If they are specified directly, an error is generated. If you are querying a slot and the slot does not contain any channels with these types, an error is generated.

Example

print(channel.getmode("slot1"))

Query the configuration of the channels on slot 1. Assuming a 7706, channels 21 and 22 are digital I/O, 23 and 24 are DAC, and channel 25 is a totalizer.

Output:

[1]=1, [2]=1, [3]=17, [4]=17, [5]=2
