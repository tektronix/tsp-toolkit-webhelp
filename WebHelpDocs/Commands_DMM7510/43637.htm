acal.nextrun.time

This attribute returns the date and time when the next autocalibration is scheduled to be run.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Attribute (R)

Yes

Not applicable

Not applicable

Not applicable

Usage

dateTime = acal.nextrun.time

dateTime

The date and time when autocalibration is scheduled to be run

Details

The date and time is returned in the format:

MM/DD/YYYY HH:MM:SS.NNNNNNNNN

Where:

  • MM/DD/YYYY is the month, date, and year
  • HH:MM:SS.NNNNNNNNN is the hour, minute, second, and fractional second

Example

print(acal.nextrun.time)

Returns date and time when autocalibration is next scheduled to be run.

Example output:

05/29/2014 17:11:17.000000000
