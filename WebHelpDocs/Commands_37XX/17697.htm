maxNumber

A variable assigned the value of the constant; the constant equals the maximum number available for the specified feature on the local node instrument

Details

These read-only constants indicate the maximum number of each feature type that is available. Feature types are timers, digital input/output lines, triggers, and blenders.

When using this command from a remote node, localnode should be replaced with the node reference, for example, node[5].define.MAX_TIMERS.

Example

maxNumber = localnode.define.MAX_TIMERS

Reads the maximum number of timers that are available for the presently active instrument.
