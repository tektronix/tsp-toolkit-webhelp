dmm.calibration.ac()

This function begins the specified AC adjustment step on the DMM.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Function

Yes

 

 

 

Usage

dmm.calibration.ac(step)

dmm.calibration.ac(step, value)

step

The AC adjustment step to perform

value

The value for this adjustment step (if the adjustment step has a value)

Details

This command generates an error if the:

  • Calibration is locked
  • Step is out of sequence
  • Step does not exist
  • Step does not complete successfully
  • Value passed is invalid for the step, out of range, or not needed

Example

For detail on how to use dmm.calibration.ac(), see AC volts adjustment, AC current adjustment, and Frequency adjustment.
