beeper.enable

This attribute allows you to turn the beeper on or off.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Attribute (RW)

Yes

Reset

Create configuration script

Enable

Usage

state = beeper.enable

beeper.enable = state

state

beeper.OFF or 0: Beeper disabled

beeper.ON or 1: Beeper enabled

Details

Disabling the beeper also disables front panel key clicks.

Example

beeper.enable = beeper.ON

beeper.beep(2, 2400)

Enables the beeper and generates a twosecond, 2400 Hz tone
