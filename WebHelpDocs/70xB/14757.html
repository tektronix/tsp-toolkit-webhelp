eventlog.count

This attribute gets the number of events contained in the event log.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Attribute (R)

Yes

System reset

Clearing event log

Reading event log

Not applicable

Not applicable

Usage

N = eventlog.count

N

The number of events in the event log

Example

print(eventlog.count)

Display the present number of events contained the TSP event log.

 

Output looks similar to:

3.00000e000
