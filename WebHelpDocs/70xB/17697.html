maxNumber

A variable assigned the value of the constant. The constant equals the local node instrument's maximum number available for the specified feature.

Details

These read-only constants to the following types of features: timers, digital input/output lines, triggers, and blenders. They provide the number of features available (which is the maximum) for the specified local node feature.

Example 1

maxNumber = localnode.define.MAX_TIMERS

Reads the maximum number of timers that are available for the presently active instrument.
