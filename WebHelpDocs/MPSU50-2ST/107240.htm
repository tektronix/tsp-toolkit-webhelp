slot[Z].psu[X].source.rangev

This attribute contains the value of the source range for voltage.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Attribute (RW)

Yes

 

 

 

Usage

value = slot[Z].psu[X].source.rangev

value

Voltage range: 50 V

Z

Module slot number

X

Module channel number

Details

This setting contains the fixed range setting for the voltage source. The range parameter can be any valid source range value for the instrument and the best source range for that level will be used.

Example

slot[1].psu[2].source.rangev = 20

range = slot[1].psu[2].source.rangev

Sets source range to 50 V given a value of 20 V for channel 2 of the module installed in slot 1.
