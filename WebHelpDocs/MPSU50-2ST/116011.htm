bufferVar.n

This attribute contains the number of readings in the buffer.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Attribute (R)

Yes

Not applicable

Not applicable

Not applicable

Usage

numberOfReadings = bufferVar.n

numberOfReadings

The number of readings stored in the buffer

bufferVar

The reading buffer; can be a dynamically allocated user-defined buffer or a default reading buffer

Details

You can use this to specify the final reading in a buffer.

Example

buffer1 = slot[1].psu[1].makebuffer(100)

buffer1.clear()

slot[1].psu[1].measure.count = 20

slot[1].psu[1].measure.v(buffer1)

printbuffer(1, buffer1.n, buffer1.readings)

Clear the buffer.

Set the measure count to 20.
