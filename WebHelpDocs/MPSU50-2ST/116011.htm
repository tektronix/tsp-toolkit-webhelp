bufferVar.n

This attribute contains the number of readings in the buffer.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Attribute (R)

Yes

Not applicable

See Details

Not applicable

Usage

numberOfReadings = bufferVar.n

numberOfReadings

The number of readings stored in the buffer

bufferVar

The reading buffer; can be a dynamically allocated user-defined buffer or a dedicated reading buffer

Details

This read-only attribute contains the number of readings in the buffer.

Example

buffer1 = slot[1].psu[1].makebuffer(100)

numberOfReadings = buffer1.n

Sets the numberOfReadings variable to the number of readings stored in buffer1.n of channel 1 of the module installed in slot 1.
