slot[Z].psu[X].measure.count

This attribute sets the number of measurements performed when a measurement is requested.

Type

TSP-Link accessible

Affected by

Default value

Attribute (RW)

Yes

1

Usage

count = slot[Z].psu[X].measure.count

slot[Z].psu[X].measure.count = count

count

Number of measurements: 1 to 5242880

Z

Module slot number

X

Module channel number

Details

This attribute controls the number of measurements made any time a measurement is requested. When using a reading buffer with a measure command, this attribute also controls the number of readings to be stored. If the count exceeds the remaining capacity of the reading buffer, no measurements are made.

Example

slot[1].psu[1].measure.count = 100

Sets the number of measurements to be performed on channel 1 of the module installed in slot 1.
