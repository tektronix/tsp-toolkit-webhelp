slot[Z].psu[X].source.protect.enablev

This attribute enables or disables the output overvoltage protection function.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Attribute (RW)

Yes

Instrument reset
PSU reset
Recall setup

Not saved

slot[Z].psu[X].ENABLE

Usage

protectEnabled = slot[Z].psu[X].source.protect.enablev

slot[Z].psu[X].source.protect.enablev = protectEnabled

protectEnabled

The state of overvoltage protection:

  • Disable: slot[Z].psu[X].DISABLE
  • Enable: slot[Z].psu[X].ENABLE

Z

Module slot number

X

Module channel number

Details

When overcurrent protection is enabled, the power supply turns off when the output current exceeds the current protection level attribute.

Example

slot[1].psu[2].source.protect.enablev = slot[1].psu[2].DISABLE

Disables overvoltage protection for channel 2 of the module installed in slot 1.
