slot[Z].psu[X].source.output

This attribute enables or disables the source output.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Attribute (RW)

Yes

Power cycle
Mainframe reset
Module reset

Not applicable

slot[Z].psu[X].OFF

Usage

value = slot[Z].psu[X].source.output

slot[Z].psu[X].source.output = value

value

State of the source output:

  • Enable: slot[Z].psu[X].ON
  • Disable: slot[Z].psu[X].OFF

Z

Module slot number

X

Module channel number

Details

When this attribute is read, it returns the output state of the source. When this attribute is set, it switches the output of the source on or off. The output voltage changes according to the slew rate set by slot[Z].psu[X].source.slewratev.

Set this attribute to slot[Z].psu[X].OFF to place the output in a high‑impedance state.

The source output setting is not stored in configuration lists.

Example

slot[1].psu[2].source.output = slot[1].psu[2].ON

Turns on the output for channel 2 of the module installed in slot 1.
