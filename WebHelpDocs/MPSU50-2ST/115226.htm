slot[Z].psu[X].configlist.table()

This function returns the names of measure configuration lists stored on the instrument.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Function

Yes

 

 

 

Usage

slot[Z].psu[X].configlist.table()

Z

Module slot number

X

Module channel number

Details

You can use this command to retrieve the names of configuration lists that are stored in the instrument.

To see the names of the configuration lists that have been created, use either of the code examples under Example 1 and Example 2. If no configuration lists exist, the table will be empty.

Example 1

myConfigList = slot[3].psu[1].configlist.table()

for i, name in pairs(myConfigLists) do

print(name)

end

Requests the name of one configuration list that is stored in the instrument. Send the command again until it returns nil to get all stored lists.

Example 2

myConfigLists = slot[3].psu[1].configlist.table()

for i = 1, myConfigLists.getn do

print(myConfigLists[i])

end

Requests the name of one configuration list that is stored in the instrument. Send the command again until it returns nil to get all stored lists.

Also see

"Configuration lists" in the General operation section of the MP5000 Modular Precision Test System Reference Manual
