bufferVar.basetimestampfractional

This attribute contains the fractional seconds portion of the timestamp that indicates when the first reading was stored in the reading buffer.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Attribute (R)

Yes

Not applicable

Not applicable

Not applicable

Usage

fractionalSec = bufferVar.basetimestampfractional

fractionalSec

The fractional second portion of the timestamp

bufferVar

The name of the reading buffer, which may be a default buffer or a user-defined buffer

Details

This read-only attribute is an array of the fractional portion of the timestamp, in seconds, when each reading occurred. Seconds are shown as fractions.

Example

reset()

buffer1 = slot[1].psu[1].makebuffer(5)

buffer1.clear()

slot[1].psu[1].measure.count = 3

slot[1].psu[1].measure.v(buffer1)

printbuffer(1, buffer1.n, buffer1.readings)

print(buffer1.basetimestampseconds .. buffer1.basetimestampfractional)

Clear the buffer.

Set the measurement count to 3.

Set up voltage measurements to go into buffer1.

Return the readings from buffer1.

Return the seconds and fractional seconds.
