slot[Z].psu[X].trigger.measure.Y()

This function configures the measurements to be made during trigger model operation and identifies the reading buffer that stores the measurements.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Function

Yes

 

 

 

Usage

slot[Z].psu[X].trigger.measure.Y(ReadingBuffer)

slot[Z].psu[X].trigger.measure.iv(CurrentBuffer, VoltageBuffer)

Z

Module slot number

X

Module channel number

Y

The measurement function; v = voltage, i = current, r = resistance, iv = current and voltage, p = power

ReadingBuffer

Buffer to store the measurement result

CurrentBuffer

Buffer to store current readings

VoltageBuffer

Buffer to store voltage readings

Details

This command configures the measurements to be made during the trigger model. After configuring the measurements to be made, add a measure block to your trigger model to perform the measurements. Any measurements made in the trigger model are stored in this reading buffer according to the specified measurement type.

Example

PsuCh1 = slot[1].psu[1]

 

--- Configure PSU Ch1 to measure voltage and save the result in the reading buffer.

ReadingBuffer = PsuCh1.makebuffer(10)

PsuCh1.trigger.measure.v(ReadingBuffer)

 

--- Set the initial voltage output for PSU Ch1 to 2V

PsuCh1.source.levelv = 2

 

--- Delete old trigger model instances before creating new ones ---

slot[1].trigger.model.delete("TM1")

slot[1].trigger.model.create("TM1")

 

--- Make the voltage measurement 10 times ---

slot[1].trigger.model.addblock.source.output("TM1", "OutputOn", 1, 1)

slot[1].trigger.model.addblock.measure("TM1", "MeasureVoltage", 1, 10)

slot[1].trigger.model.addblock.source.output("TM1", "OutputOff", 1, 0)

 

--- Execute the trigger model ---

slot[1].trigger.model.initiate("TM1")

waitcomplete()

-- Now printbuffer the 10 measurements.

printbuffer(1, ReadingBuffer.n, ReadingBuffer)

A reading buffer is configured for voltage measurements and a measure block captures measurements for the module installed in slot 1.
