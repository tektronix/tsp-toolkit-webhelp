Connecting to the TSP-Link system

The TSP‑Link® trigger lines are built into the TSP-Link connection. Use the TSP-Link connectors on the back of the 2600B. If you are using a TSP‑Link network, you do not have to modify any connections. See TSP-Link system expansion interface for detailed information about connecting to the TSP-Link system.
