Tools for managing scripts

To manage scripts, you can send messages to the instrument, use your own development tool or program, use Keithley Instruments Test Script Builder (TSB) software, or use TSB Embedded on the web interface of the instrument.

TSB software is a programming tool that you can download from the Product Support and Downloads web page. You can use it to create, modify, debug, and store Test Script Processor (TSP®) scripting engine scripts. For more information about using the TSB software, see Using Test Script Builder (TSB).

TSB Embedded is a tool with a reduced set of features than the complete Keithley TSB software. TSB Embedded has both script-building functionality and console functionality (single‑line commands). It is accessed from a web browser. Refer to Working with TSB Embedded for additional information.

If you are using TSB or TSB Embedded to create scripts, you do not need to use the commands loadscript or loadandrunscript and endscript.
