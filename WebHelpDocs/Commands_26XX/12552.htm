Action overruns on hardware lines

An action overrun occurs when a trigger event is received before the digital I/O or TSP‑Link® line is ready to process it. The generation of an action overrun is dependent upon the trigger mode selected for that line. For more details on the causes of action overruns, see Hardware trigger modes. Use the status model to monitor for the occurrence of action overruns. For details, see Status model.
