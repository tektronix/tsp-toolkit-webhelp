Using TSP-Link trigger lines for digital I/O

Each trigger line is an open-drain signal. When using the TSP‑Link® trigger lines for digital I/O, any node that sets the programmed line state to zero (0) causes all nodes to read 0 from the line state. This occurs regardless of the programmed line state of any other node. Refer to the table in Digital I/O bit weighting for digital bit‑weight values.
