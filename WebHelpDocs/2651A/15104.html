smuX.contact.threshold

This attribute stores the resistance threshold for the smuX.contact.check() function.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Attribute (RW)

Yes

Instrument reset
SMU reset
Recall setup

Not saved

50 (50 W)

Usage

rValue = smuX.contact.threshold

smuX.contact.threshold = rValue

rValue

The resistance above which contact check should fail (measured in ohms)

X

SMU channel (for example, smua.contact.threshold applies to SMU channel A)

Details

The threshold should be set to less than 1 kW.

Example

smua.contact.threshold = 5

Set the contact check threshold for SMU channel A to 5 W.
