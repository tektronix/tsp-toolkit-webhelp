localnode.model

This attribute stores the model number.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Attribute (R)

Yes

Not applicable

Not applicable

Not applicable

Usage

model = localnode.model

model

The model number of the instrument

Details

When using this command from a remote node, localnode should be replaced with the node reference, for example node[5].model.

Example

print(localnode.model)

Outputs the model number.
