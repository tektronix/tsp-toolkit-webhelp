localnode.serialno

This attribute stores the instrument's serial number.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Attribute (R)

Yes

Not applicable

Not applicable

Not applicable

Usage

serialno = localnode.serialno

serialno

The serial number of the instrument

Details

This read-only attribute indicates the instrument serial number.

When using this command from a remote node, localnode should be replaced with the node reference, for example node[5].serialno.

Example

display.clear()

display.settext(localnode.serialno)

Clears the instrument's display.

Places the instrument's serial number on the top line of its display.
