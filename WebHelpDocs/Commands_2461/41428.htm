smu.digitize.func

This attribute determines which digitize function is active.

smu.FUNC_NONE

Usage

value = smu.digitize.func

smu.digitize.func = value

value

The digitize measurement function to make active:

  • Current: smu.FUNC_DIGITIZE_CURRENT
  • Voltage: smu.FUNC_DIGITIZE_VOLTAGE
  • No digitize function selected (read only): smu.FUNC_NONE

Details

Set this command to the type of measurement you want to digitize.

Reading this command returns the digitize function that is presently active.

If a basic (non-digitize) measurement function is selected, this returns smu.FUNC_NONE. The none setting is automatically made if you select a function with smu.measure.func or through the options from the front-panel Measure Functions tab.

Example

smu.digitize.func = smu.FUNC_DIGITIZE_CURRENT

Set the measurement function to digitize current.
