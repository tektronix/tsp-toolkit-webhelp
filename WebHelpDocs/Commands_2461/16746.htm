What can be included in scripts?

Scripts can include combinations of Test Script Processor (TSP®) commands and Lua code. TSP commands instruct the instrument to do one thing and are described in the command reference (see TSP commands). Lua is a scripting language that is described in Fundamentals of programming for TSP.
