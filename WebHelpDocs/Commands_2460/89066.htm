available()

This function checks for the presence of specific instrument functionality.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Function

No

 

 

 

Usage

presence = available(functionality)

presence

If the functionality is present, returns true; if not, returns false

functionality

The functionality to check for:

  • Digital I/O: digio
  • GPIB communications: gpib
  • TSP-Link: tsplink

Example

print(available(gpib))

Returns true if GPIB communications are available. Returns false if GPIB communications are not available.
