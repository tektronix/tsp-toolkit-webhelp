Test Script Builder (TSB)

Keithley Instruments Test Script Builder (TSB) is a software tool you can use to develop scripts for TSP-enabled instruments.

You must use the TSP command set to use Test Script Builder. Refer to Determining the command set you will use for information about the command sets and changing them.
