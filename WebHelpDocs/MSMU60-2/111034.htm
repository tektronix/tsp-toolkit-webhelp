slot[Z].smu[X].source.delay

This attribute contains the delay that occurs when the source is changed.

Type

Usage

sDelay = slot[Z].smu[X].source.delay

slot[Z].smu[X].source.delay = sDelay

sDelay

The length of the delay in seconds

Z

Module slot number

X

Module channel number

Details

This attribute allows for additional delay (settling time) after an output step. Set sDelay to a user-defined value (in seconds). Delays are only applied when the source output is on.

Example

slot[1].smu[1].source.delay = 3e-3

Sets a 3 ms delay after each source action while the source is turned on for channel 1 of the module installed in slot 1.
