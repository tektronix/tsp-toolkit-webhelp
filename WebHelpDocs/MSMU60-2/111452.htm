slot[Z].smu[X].source.offlimitY

This attribute configures both the positive and negative source off limit settings symmetrically.

Type

Affected by

Where saved

Default value

Attribute (RW)

Yes

Voltage: 200e−3 V
Current: 1e−6 A

Usage

value = slot[Z].smu[X].source.offlimitY

slot[Z].smu[X].source.offlimitY = value

value

MSMU60-2

  • Voltage: 0 V to 60 V
  • Current: 0 A to 1.5 A

Z

Module slot number

X

Module channel number

Y

The measurement function; v = voltage, i = current

Details

This setting configures both the positive and negative source off limit settings symmetrically.

Example

slot[1].smu[1].source.offlimiti = 35e-3

Sets the positive current off limit to 35 mA and the negative current off limit to −35 mA.
