slot[Z].smu[X].source.autorangeY

This attribute contains the state of the source autorange control (on or off).

Type

TSP-Link accessible

Affected by

Where saved

Default value

Attribute (RW)

Yes

Instrument reset
SMU reset

Not applicable

slot[Z].smu[X].ON

Usage

sourceAutorange = slot[Z].smu[X].source.autorangeY

slot[Z].smu[X].source.autorangeY = sourceAutorange

sourceAutorange

slot[Z].smu[X].OFF

slot[Z].smu[X].ON

Z

Module slot number

X

Module channel number

Y

The measurement function; v = voltage, i = current

Details

If voltage autorange is enabled and you attempt to change the voltage level while overlapped measurements are in progress, an error is generated and the setting is not applied. If current autorange is enabled and you attempt to change the current level while overlapped measurements are in progress, an error is generated and the setting is not applied.

When autorange is on, the source range value is set to the autorange value.

Example

slot[1].smu[2].source.autorangev = slot[1].smu[2].OFF

slot[1].smu[2].source.rangev = 20

Disables the autorange feature for channel 2 of the module installed in slot 1 and set a fixed source range of 20 V.
