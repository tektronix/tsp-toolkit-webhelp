bufferVar.timestamps

This attribute contains the timestamp for each reading in the specified buffer.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Attribute (R)

Yes

Not applicable

Not applicable

Not applicable

Usage

timestamp = bufferVar.timestamps[N]

timestamp

The complete timestamp (including date, time, and fractional seconds) of reading number N in the specified reading buffer when the reading was acquired

bufferVar

The reading buffer can be a dynamically allocated user-defined buffer or a default reading buffer

N

The reading number: 1 to bufferVar.n

Details

This buffer recall attribute is similar to an array (a Lua table) that contains UTC timestamp of when each reading occurred.

Example

buffer1 = slot[1].smu[1].makebuffer(100)

timestamp = buffer1.timestamps[1]

Retrieve the timestamp of the first reading stored in buffer1.
