slot[Z].smu[X].contact.r()

This function performs a contact check measurement and returns the measured contact resistances.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Function

Yes

 

 

 

Usage

hi, lo = slot[Z].smu[X].contact.r()

hi

The contact check hi measurement in ohms

lo

The contact check lo measurement in ohms

Z

Module slot number

X

Module channel number

Example

print(slot[1].smu[1].contact.r())

Performs a measurement of the hi and lo contact resistances for channel 1 of the module installed in slot 1.
