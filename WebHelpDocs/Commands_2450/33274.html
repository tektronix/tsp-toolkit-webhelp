Reading an attribute

To read an attribute, you can use the attribute as the parameter of a function or assign it to another variable.

Example 3: Read an attribute using a function

print(beeper.enable)

Reads the status of the beeper by passing the attribute to the print function. If the beeper is set on, the output is:

beeper.ON

Example 4: Read an attribute using a variable

beep = beeper.enable

print(beep)

This reads the beeper status by assigning the attribute to a variable named beep. If the beeper is set off, the output is:

beeper.OFF
