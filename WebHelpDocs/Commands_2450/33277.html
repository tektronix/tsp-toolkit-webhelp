Commands that cannot be used in scripts

Though an instrument accepts the following commands, you cannot use these commands in scripts.

General commands that cannot be used in scripts:

  • abor
  • abort
  • endflash
  • endscript
  • flash
  • loadscript
  • loadandrunscript
  • login
  • logout
  • prevflash

Common commands that cannot be used are shown in the following table with equivalent commands that can be used in scripts.
