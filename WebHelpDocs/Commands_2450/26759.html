Tools for managing scripts

You can use any of the following tools to manage scripts:

  • The front‑panel menu options and flash drive. For information, see Saving setups in the Reference Manual.
  • Messages sent to the instrument. For information, see Load a script by sending commands over a remote interface.
  • Keithley Instruments Test Script Builder (TSB) software (included on the Test Script Builder Software Suite CD-ROM that was included with your instrument). For more information, see Creating a new TSP project.
  • Your own development tool or program.
