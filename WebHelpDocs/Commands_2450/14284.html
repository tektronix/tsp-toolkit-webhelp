Lua basics

This section contains the basics about the Lua programming language to allow you to start adding Lua programming commands to your scripts quickly.

For more information about Lua, see the Lua website (http://www.lua.org). Another source of useful information is the Lua users group (http://lua-users.org), created for and by users of Lua programming language.
