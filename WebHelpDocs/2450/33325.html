Function and variable name restrictions

You cannot use Lua reserved words and top level command names for function or variable names.

You cannot use the following Lua reserved words for function or variable names.

Lua reserved words

and

for

or

break

function

repeat

do

if

return

else

in

then

elseif

local

true

end

nil

until

false

not

while

There are additional words you cannot use as variable names. These include top‑level command names; if you use these names, it results in the loss of use of the commands. For example, if you send the command digio = 5, you cannot access the digio.* commands until you turn the instrument power off and then turn it on again.
