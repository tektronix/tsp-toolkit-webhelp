Autoexec script

The autoexec script runs automatically when the instrument is turned on. To create an autoexec script, you must create the script and save it to nonvolatile memory.

To create the autoexec script, save a new script that is named autoexec. See Saving a script using the remote interface.

To save the script to nonvolatile memory, send the command:

autoexec.save()
