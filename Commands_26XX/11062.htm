RS-232 interface operation

The following topics contain information about configuring RS-232 communication parameters, sending or receiving command messages, and requesting or retrieving data. To control the 2600B, connect a controller or personal computer to the 2600B RS-232 interface. Alternatively, you can use the 2600B to control another device over RS-232.
