eventlog.next()

This function returns the oldest message from the event log and removes it from the log.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Function

Yes

 

 

 

Usage

logString = eventlog.next()

logString

The next log entry

Details

Returns the next entry from the event log and removes it from the log.

If there are no entries in the event log, returns the value nil.

Example 1

print(eventlog.next())

Output:

17:28:22.085 10 Oct 2009, LAN2, 192.168.1.102, LXI, 0, 1192037134, <no time>, 0, 0x0

Example 2

print(eventlog.next())

Output:

17:28:25.549 10 Oct 2009, LAN6, 192.168.1.102, LXI, 0, 1192037135, <no time>, 0, 0x0

Example 3

print(eventlog.next())

Output:

17:28:31.563 10 Oct 2009, LAN4, 192.168.1.102, LXI, 0, 1192037136, <no time>, 0, 0x0

Example 4

print(eventlog.next())

Output:

nil
