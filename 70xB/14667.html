digio functions and attributes

The digital I/O port of the TSP can control external circuitry (such as a component handler for binning operations). The I/O port has 14 lines. Each line can be at TTL logic state 1 (high) or 0 (low). See the pinout in Rear panel summary for additional information.

There are commands to read and write to each individual bit, and commands to read and write to the entire port. Use the following code fragment to write to one bit of the digital I/O port. The I/O bit is then read and the state is returned to the computer, where it can be processed or displayed.

Example: Write to one bit of the digital I/O port

Command

Description

digio.writebit(4, 0)

Writes a 0 to Bit 4

data = digio.readbit(4)

Reads the value at Bit 4

print(data)

Generates a response message on the computer
