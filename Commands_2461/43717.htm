smu.digitize.unit

This attribute sets the units of measurement that are displayed on the front panel of the instrument and stored in the reading buffer.

Type

TSP-Link accessible

Affected by

Where saved

Default value

Attribute (RW)

Yes

Restore configuration
Instrument reset
Power cycle
Measure configuration list

Configuration script
Measure configuration list

smu.UNIT_VOLT (digitize voltage function)
smu.UNIT_AMP (digitize current function)

Usage

value = smu.digitize.unit

smu.digitize.unit = value

value = smu.measure.getattribute(function, smu.ATTR_DIGI_UNIT)

smu.measure.setattribute(function, smu.ATTR_DIGI_UNIT, value)

value

Units to display for the digitize voltage function:

  • Volts: smu.UNIT_VOLT
  • Ohms: smu.UNIT_OHM
  • Watts: smu.UNIT_WATT

    Units to display for the digitize current function:

  • Amps: smu.UNIT_AMP
  • Ohms: smu.UNIT_OHM
  • Watts: smu.UNIT_WATT

function

The digitize function:

  • Current: smu.FUNC_DIGITIZE_CURRENT
  • Voltage: smu.FUNC_DIGITIZE_VOLTAGE

Details

The change in measurement units is displayed when the next measurement is made.

Example

smu.digitize.func = smu.FUNC_DIGITIZE_VOLTAGE

smu.digitize.unit = smu.UNIT_WATT

Set the measure function to digitize voltage.

Set the units to display as watts.
